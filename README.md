# React Hierarchical Allocation Table

This project implements a hierarchical allocation table with:

- Parent / child rollups
- Allocation by percentage and value
- Variance calculation based on original values
- Automatic subtotal updates
- Grand total calculation

## Tech Stack
- React
- TypeScript
- CSS

## Features
- Recursive tree rendering
- Immutable state updates
